Adding a custom filter

At this point, the autocomplete panel should be toggleable on focus and options should be selectable. But if we want our options to filter when we type, we need to add a custom filter.

Filter autocomplete

Setting separate control and display values

If you want the option's control value (what is saved in the form) to be different than the option's display value (what is displayed in the text field), you'll need to set the displayWith property on your autocomplete element. A common use case for this might be if you want to save your data as an object, but display just one of the option's string properties.


Automatically highlighting the first option

If your use case requires for the first autocomplete option to be highlighted when the user opens the panel, you can do so by setting the autoActiveFirstOption input on the mat-autocomplete component. This behavior can be configured globally using the MAT_AUTOCOMPLETE_DEFAULT_OPTIONS injection token.


Autocomplete on a custom input element

While mat-autocomplete supports attaching itself to a mat-form-field, you can also set it on any other input element using the matAutocomplete attribute. This allows you to customize what the input looks like without having to bring in the extra functionality from mat-form-field.

Option groups

mat-option can be collected into groups using the mat-optgroup element:


        <mat-autocomplete #autoGroup="matAutocomplete">
         <mat-optgroup *ngFor="let group of stateGroupOptions | async" [label]="group.letter">
            <mat-option *ngFor="let name of group.names" [value]="name">
              {{}}
              </mat-option>
          </mat-optgroup>
        </mat-autocomplete>

          // Angular
          import  { Component, ViewEncapsulation, OnInit, Injector, ViewChild, ElementRef, forwardRef, Input, Renderer2, AfterViewInit } from '@angular/core';
          import  { NG_VALUE_ACCESSOR, } from '@angular/forms';

          // SyncFusion
          import  { SpeedDialComponent, SpeedDialItemModel } from '@syncfusion/ej2-angular-buttons';

          // Virtue
          import  { ValueAccessor } from '@virtuetec/lib/core';

          @Component( {
            selector: 'virtue-speeddial',
            templateUrl: './buttonspeeddial.component.html',
            styleUrls: ['./buttonspeeddial.component.scss'],
            encapsulation: ViewEncapsulation.None,
            providers: [ {
              provide: NG_VALUE_ACCESSOR,
              useExisting: forwardRef(() => ButtonSpeedDialComponent),
              multi: true
            }]
          })
          export class ButtonSpeedDialComponent extends ValueAccessor implements OnInit, AfterViewInit  {

            //#region Constructor

            constructor(
              protected _el: ElementRef,
              protected _injector: Injector,
              protected _renderer: Renderer2)  {
              super(_el, _injector, _renderer);
            }

            //#endregion

            //#region AfterViewInit

            ngAfterViewInit(): void  {

            }

            //#endregion

            //#region Implementation OnInit

            ngOnInit(): void  {
              if (this.control != null)
                this.control.disable();
            }

            //#endregion

            //#region Input Properties


            //#endregion

            //#region Public Methods

            public closeClick()  {
              this.speeddialObj.hide();
            }

            public styleObject(): Object  {
              return  { 'margin-bottom': this.marginBottom, 'margin-right': this.marginRight };
            }

            public submitClick()  {
              this.speeddialObj.hide();
            }

            //#endregion

            //#region View Objects

            @Input('headTextModal') headTextModal: string = 'HEAD NÃO INFORMADO';
            @Input('iconButton') iconButton: string = 'e-table-of-content';
            @Input('itemsSource') items: SpeedDialItemModel[] = [];
            @Input('labelText') labelText: string = 'LABEL NÃO INFORMADO';
            @Input('marginBottom') marginBottom: string = "75px";
            @Input('marginRight') marginRight: string = "10px";

            @ViewChild('popupSpeeddial') public speeddialObj: SpeedDialComponent;

            //#endregion
          }
